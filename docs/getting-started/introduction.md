# Introduction

Netomi is an AI-powered customer service platform that resolves support tickets automatically across email, chat, SMS, and voice channels.

This documentation is for developers integrating Netomi into existing support workflows, building custom automations, or extending the platform through the API.

## What you can do with Netomi

- Automate ticket resolution across multiple channels
- Connect Netomi to your CRM, helpdesk, or internal systems
- Build custom AI workflows using the REST API
- Monitor resolution rates, escalation paths, and model performance

## Who this is for

This documentation assumes you are a developer or technical team member responsible for implementing or maintaining a Netomi integration. Basic familiarity with REST APIs and JSON is expected.

## How to use these docs

Start with the [Quickstart](./quickstart) to get a working integration in under 15 minutes. Then read [Core Concepts](../guides/core-concepts) to understand how the platform works before building anything production-ready.
